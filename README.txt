Welcome to Tic Tac Toe Game!

This repository contains the source code for a classic Tic Tac Toe game implemented using HTML, CSS, and JavaScript.

## Features
- Simple and intuitive gameplay.
- Play against a friend.
- Responsive design for seamless gameplay on different devices.

## Usage
1. Clone this repository to your local machine.
2. Open the `index.html` file in your web browser.
3. Click on the grid to place your X or O.
4. Aim to get three in a row horizontally, vertically, or diagonally to win the game.
5. Enjoy the game!
